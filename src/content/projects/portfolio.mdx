While this may seem like your common, everyday portfolio website, there is some interesting stuff under the hood. The public website is published as an static site using Astro and there are some components that are rendered using hydration and Javascript islands (such as the image carousel). There is also an admin dashboard where I am able to create/edit the posts of this website, also manage the media (images, files, etc.), which allows me to blog-on-the-go!
