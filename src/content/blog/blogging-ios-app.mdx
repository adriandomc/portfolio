When I created this website I had one thing in mind: be consistent. By which I mean that I want to keep this blog updated constantly. To achieve that I looked for mobile apps that I could use to "blog on the go", but I couldn't find an all-in-one solution that would allow me to directly create/modify files on my Github repo and push changes seamlessly.

Enter **Dinoblogger**: a mobile app designed with the sole purpose of creating MDX and TXT files with ease and pushing them into my remote repo.
